My Movies App

Description

MyMovies-client is a client side application built using ReactJS for its existing server-side code (myMovies). It is a responsive app that displays a list of movies for registered users. Users can create their own account, view the movie details, search by title and select favorite movies.

Objectives

MyMovies-client is used when making requests to and receiving responses from its server-side myMovies. The application is written using the React library in ES2015+ and React Redux (hence respecting the Flux pattern). MyMovies-client uses Bootstrap as a UI library for styling and responsiveness

Meet App

Description

The Meet-app shows events for web developers in several cities around the world (ex. Berlin, Milan, Sydney). Programmers can search events by city and select how many events per page they wish to see. By using charts, the app aids users to visualize event according to their topic and city they take place.

Objectives

Meet-app app is a serverless and progressive web application (PWA). It is also build using a test-driven development (TDD) technique, which tests before writing the actual functionality of the app in code. The application uses the Meetup API to fetch upcoming events for a selected city.

List of Technologies

React
Meetup API and OAuth2 authentication flow
AWS lambda
Lighthouse PWA checklist
